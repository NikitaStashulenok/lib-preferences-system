# Use Cases (высокоуровневые)

Ниже 21 use case (требование: не менее 15).

1. Регистрация пользователя.
2. Вход пользователя с выдачей JWT.
3. Обновление access token по refresh token.
4. Выход и отзыв refresh token.
5. Просмотр каталога книг (пагинация/сортировка/фильтрация).
6. Поиск книг по названию/автору/жанру.
7. Просмотр карточки книги.
8. Добавление книги библиотекарем.
9. Редактирование книги библиотекарем.
10. Выдача книги пользователю.
11. Возврат книги пользователем.
12. Продление срока выдачи (при политике разрешения).
13. Оценка книги (rating).
14. Оставление текстового отзыва.
15. Получение персональных рекомендаций.
16. Обновление профиля предпочтений (явно пользователем).
17. Импорт/обогащение метаданных из внешнего API.
18. Администрирование пользователей и ролей.
19. Бронирование книги при отсутствии доступных экземпляров.
20. Отмена бронирования пользователем.
21. Уведомление пользователя о доступности забронированной книги (статус `NOTIFIED`, окно подтверждения 24 часа).
